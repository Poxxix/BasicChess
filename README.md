# Cờ Vua 2 Người Sử Dụng Socket  :black_square_button:

## Giới Thiệu

Project này là một ứng dụng cờ vua cho hai người chơi, được phát triển bằng ngôn ngữ Java và sử dụng socket để kết nối hai người chơi qua mạng. Ứng dụng được xây dựng và phát triển trong môi trường Eclipse.

## Các Tính Năng Chính

- **Chơi Cờ Vua 2 Người:** Hai người chơi có thể tham gia vào một ván cờ trực tuyến.
- **Giao Tiếp Qua Socket:** Sử dụng socket để truyền thông tin giữa hai người chơi.
- **Giao Diện Người Dùng:** Giao diện đồ họa đơn giản, dễ sử dụng.

## Cài Đặt

1. **Yêu Cầu:**
   - Java Development Kit (JDK)
   - Eclipse IDE

2. **Hướng Dẫn Cài Đặt:**
   - Tải và cài đặt JDK và Eclipse IDE từ trang chủ của Oracle và Eclipse.
   - Tải mã nguồn của project từ [đây](#).
   - Mở Eclipse và import project vào không gian làm việc.
   - Chạy ứng dụng và bắt đầu chơi cờ vua .

## Sử Dụng

- Mở project trong Eclipse.
- Chạy server socket để chờ kết nối từ người chơi khác.
- Người chơi thứ hai kết nối tới server qua địa chỉ IP.
- Bắt đầu chơi cờ vua.
